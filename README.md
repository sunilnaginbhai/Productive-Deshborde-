# 📖 Overview

Productive Dashboard is a clean, responsive, and feature-rich productivity application designed to improve focus and task management. It combines modern UI design with essential productivity tools to create a distraction-free workspace.

The project emphasizes performance, usability, responsive layouts, and smooth user interactions while following modern frontend development practices.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/sunilnaginbhai/Productive-Deshborde.git
```

## Navigate

```bash
cd Productive-Deshborde
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```
